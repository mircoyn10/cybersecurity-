Documentazione per il Software Universal Keydumper
Introduzione

Questo software è stato progettato per infiltrarsi nei sistemi e catturare informazioni sensibili come password e credenziali degli utenti. Utilizzando tecniche di keylogging, il software è in grado di monitorare l'attività dell'utente e inviare i dati raccolti a un server remoto per l'analisi successiva.

Requisiti di Sistema

Il software è progettato per funzionare su una varietà di sistemi operativi, inclusi Windows e Linux. È richiesto l'accesso privilegiato per eseguire il software su Windows al fine di accedere ai file di sistema sensibili.
Installazione

    Scarica il codice sorgente dal repository.
    Assicurati di avere Python installato sul tuo sistema.
    Esegui il software con privilegi elevati su Windows (clic destro sul file e seleziona "Esegui come amministratore").

Utilizzo

    Avvia il software sul sistema di destinazione.
    Il software inizierà a monitorare l'attività dell'utente, catturando password e altre informazioni sensibili.
    I dati raccolti verranno inviati a un server remoto per l'analisi successiva.
In questo script, la porta SMTP utilizzata è la porta predefinita per Gmail (587). Puoi aggiornare questa porta in base alle tue esigenze o alle specifiche del tuo provider di servizi email.

    Nella versione fast modifica il formato del file in txt, successivamente sostituisci nel codice la tua mail, con password per inviare la mail con le informazione estrapolate, e la mail di destinazione delle informazioni.
